# AGENTS.md

## 项目背景

**okstdio-enhance** 是 [okstdio](https://github.com/jianjian2048/okstdio) 项目的增强分支。

**okstdio** 是一个基于 Stdio 的 JSON-RPC 父子进程通信框架，提供了：
- JSON-RPC 2.0 协议实现
- 异步通信支持
- 中间件机制
- 自动文档生成
- 流式响应支持

## 项目定位

okstdio-enhance 的目标是在 okstdio 的基础上增加功能，使框架更加健壮。具体增强功能尚未明确，正在规划中。

## 当前状态

- **基础框架**: 基于 okstdio 0.1.0 版本
- **Python 版本**: >=3.10
- **包管理**: 使用 uv
- **开发状态**: Alpha (0.1.0)

## 技术栈

- Python 3.10+
- Pydantic 2.x
- asyncio
- uv (包管理器)
