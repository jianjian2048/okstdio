# Python

涵盖 Python 软件开发全栈指南。包括项目配置、包管理、CLI 开发和流行库。

## 结构

| 文件 | 用途 |
|------|---------|
| `SKILL.md` | Agent 技能定义（frontmatter + 指令） |
| `metadata.json` | 机器可读元数据和版本控制 |
| `AGENTS.md` | Agent 优化的快速参考（生成） |
| `README.md` | 本文件 |

## 子技能

| 技能 | 描述 |
|-------|-------------|
| [`cli/`](cli/) | 构建 Python 命令行接口和终端应用的指南。 |
| [`package-management/`](package-management/) | Python 包管理、依赖解析、虚拟环境和锁文件的指南。 |
| [`packages/`](packages/) | Python 包选择和依赖管理的指南。用于选择 Python 库、管理 pip/poetry... |
| [`project-system/`](project-system/) | 使用 pyproject.toml、构建后端和现代 PEP 标准进行 Python 项目配置和打包的指南。 |

## 使用

```bash
npx agentskills add Tyler-R-Kendrick/agent-skills/skills/python
```
