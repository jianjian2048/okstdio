# Python 项目系统

使用 pyproject.toml、构建后端和现代 PEP 标准进行 Python 项目配置和打包的指南。

## 结构

| 文件 | 用途 |
|------|---------|
| `SKILL.md` | Agent 技能定义（frontmatter + 指令） |
| `metadata.json` | 机器可读元数据和版本控制 |
| `AGENTS.md` | Agent 优化的快速参考（生成） |
| `README.md` | 本文件 |
| `rules/` | 10 个单独的最佳实践规则 |

## 使用

```bash
npx agentskills add Tyler-R-Kendrick/agent-skills/skills/python/project-system
```
