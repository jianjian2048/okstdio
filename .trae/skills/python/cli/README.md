# Python CLI 开发

Python 命令行界面和终端应用程序的构建指南。

## 结构

| 文件 | 用途 |
|------|---------|
| `SKILL.md` | Agent 技能定义（frontmatter + 指令） |
| `metadata.json` | 机器可读元数据和版本控制 |
| `AGENTS.md` | Agent 优化的快速参考（生成） |
| `README.md` | 本文件 |
| `rules/` | 10 个单独的最佳实践规则 |

## 使用

```bash
npx agentskills add Tyler-R-Kendrick/agent-skills/skills/python/cli
```
