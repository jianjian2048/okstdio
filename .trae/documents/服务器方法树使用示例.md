# 服务器方法树功能使用示例

## 功能概述

服务器方法树功能允许 RPCClient 通过内置的 `__system__` 方法获取服务器的完整方法树结构，包括：
- 所有注册的方法（名称、标签、路径、文档、参数、返回值）
- 中间件列表
- 路由器层级结构

## 服务端使用

### 自动注册

系统方法会在 RPCServer 初始化时自动注册，无需手动配置：

```python
from okstdio.server import RPCServer

app = RPCServer("my_server", label="我的服务器", version="v1.0.0")

@app.add_method(label="问候")
def hello(name: str) -> str:
    """问候方法"""
    return f"Hello, {name}!"

# 系统方法 __system__ 已自动注册
```

### 方法树结构

调用 `__system__` 方法返回的数据结构：

```json
{
    "server_name": "my_server",
    "version": "v1.0.0",
```

## 客户端使用

### 基本用法

```python
from okstdio.client import RPCClient
import asyncio

async def main():
    async with RPCClient("my_client") as client:
        # 启动服务器
        await client.start("my_server_module")
        
        # 获取服务器方法树
        method_tree = await client.get_server_methods()
        
        # 打印服务器信息
        print(f"服务器：{method_tree['server_name']}")
        print(f"版本：{method_tree['version']}")
        print(f"标签：{method_tree['label']}")
        
        # 列出所有顶层方法
        print("\n顶层方法:")
        for method in method_tree["methods"]:
            print(f"  - {method['name']}: {method['doc']}")
        
        # 列出所有路由
        print("\n路由:")
        for router_name, router_info in method_tree["routers"].items():
            print(f"  - {router_name}: {router_info['label']}")
            for method in router_info["methods"]:
                print(f"      * {method['path']}: {method['doc']}")

asyncio.run(main())
```

### 动态发现服务器能力

```python
async def discover_server_capabilities(client):
    """动态发现服务器支持的功能"""
    method_tree = await client.get_server_methods()
    
    # 检查是否支持某个方法
    all_methods = []
    for method in method_tree["methods"]:
        all_methods.append(method["path"])
    
    for router_name, router_info in method_tree["routers"].items():
        for method in router_info["methods"]:
            all_methods.append(method["path"])
    
    if "user.create" in all_methods:
        print("服务器支持用户创建功能")
        # 调用用户创建方法
        future = await client.send("user.create", {"name": "张三"})
        response = await future
        print(f"创建结果：{response.result}")
    else:
        print("服务器不支持用户创建功能")
```

### 生成客户端 SDK

```python
async def generate_client_sdk(client):
    """根据服务器方法树生成动态 SDK"""
    method_tree = await client.get_server_methods()
    
    class DynamicSDK:
        def __init__(self, client):
            self.client = client
        
        async def call(self, method, **params):
            """通用调用方法"""
            future = await self.client.send(method, params)
            response = await future
            return response.result
    
    sdk = DynamicSDK(client)
    
    # 动态添加方法
    for method in method_tree["methods"]:
        method_name = method["path"]
        
        async def make_call(m=method_name):
            async def call(**params):
                return await sdk.call(m, **params)
            return call
        
        setattr(sdk, method_name.replace(".", "_"), await make_call(method_name))
    
    return sdk
```

## 直接调用方法树

### 服务端直接获取

```python
from okstdio.server import RPCServer

app = RPCServer("my_server")

# 方式一：使用 get_method_tree() 方法
tree = app.get_method_tree()
print(tree)

# 方式二：使用 docs_json() 方法（向后兼容）

import asyncio

```

## 方法树字段说明

### 服务器级别字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| server_name | str | 服务器名称 |
| version | str | 服务器版本 |
| label | str | 服务器标签/描述 |
| methods | list | 顶层方法列表 |
| middlewares | list | 中间件列表 |
| routers | dict | 路由器字典 |

### 方法字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| name | str | 方法名称 |
| label | str | 方法标签 |
| path | str | 方法完整路径（包含路由前缀） |
| doc | str | 方法文档字符串 |
| params | list | 参数列表 |
| results | list | 返回值说明列表 |

### 参数字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| name | str | 参数名称 |
| kind | str | 参数类型（POSITIONAL_OR_KEYWORD 等） |
| required | bool | 是否必填 |
| type | str | 参数类型名称 |
| default | any | 默认值 |
| description | str | 参数描述（从 FieldInfo 提取） |

## 最佳实践

### 1. 使用标签和文档

为方法和路由添加清晰的标签和文档，方便客户端理解：

```python
@app.add_method(label="创建用户", name="user.create")
def create_user(name: str, age: int) -> dict:
    """创建新用户
    
    Args:
        name: 用户姓名
        age: 用户年龄
    
    Returns:
        创建的用户信息，包含 id、name、age 字段
    """
    return {"id": 1, "name": name, "age": age}
```

### 2. 使用 Pydantic 模型提供详细文档

```python
from pydantic import BaseModel, Field

class UserCreate(BaseModel):
    """用户创建请求模型"""
    name: str = Field(description="用户姓名", min_length=1, max_length=50)
    age: int = Field(description="用户年龄", ge=0, le=150)
    email: str | None = Field(description="邮箱地址", default=None)

@app.add_method()
def create_user_v2(data: UserCreate) -> dict:
    """创建用户（增强版）"""
    return {"id": 1, **data.model_dump()}
```

### 3. 客户端缓存方法树

避免频繁调用 `__system__` 方法，客户端应缓存方法树：

```python
class CachedClient:
    def __init__(self, client):
        self.client = client
        self._method_tree = None
    
    async def get_method_tree(self, force_refresh=False):
        if self._method_tree is None or force_refresh:
            self._method_tree = await self.client.get_server_methods()
        return self._method_tree
```

### 4. 版本兼容性检查

```python
async def check_server_compatibility(client):
    """检查服务器版本兼容性"""
    method_tree = await client.get_server_methods()
    version = method_tree["version"]
    
    # 解析版本号
    major_version = int(version.lstrip("v").split(".")[0])
    
    if major_version < 1:
        raise RuntimeError(f"不支持的服务器版本：{version}，需要 v1.x 或更高版本")
    
    return True
```

## 注意事项

1. **系统方法不经过中间件**：`__system__` 方法不经过任何中间件，保证系统功能始终可用
2. **系统方法不可覆盖**：`__system__` 方法名被保留，用户不能定义同名的方法
3. **方法树不包含系统方法**：返回的方法树中不会包含 `__system__` 本身
4. **异步安全**：所有方法树相关的方法都是异步安全的

## 故障排除

### 问题：客户端无法获取方法树

**可能原因**：
- 服务器未启动
- 网络通信问题
- 服务器版本过低（不支持 `__system__` 方法）

**解决方法**：
```python
try:
    method_tree = await client.get_server_methods()
except RuntimeError as e:
    print(f"服务器未启动：{e}")
except Exception as e:
    print(f"获取方法树失败：{e}")
    # 降级处理
```

### 问题：方法树信息不完整

**可能原因**：
- 方法没有添加文档字符串
- 参数类型注解缺失
- 路由未正确挂载

**解决方法**：
```python
# 确保添加文档字符串
@app.add_method()
def my_method(param1: str, param2: int = 10) -> dict:
    """方法文档
    
    Args:
        param1: 参数 1 说明
        param2: 参数 2 说明，默认值为 10
    
    Returns:
        返回说明
    """
    return {}
```
